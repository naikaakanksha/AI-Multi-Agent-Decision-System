# 🤖 AI Multi-Agent Decision System

## 🚀 Overview

The **AI Multi-Agent Decision System** is an intelligent application designed to simulate multiple AI agents, each analyzing a decision from a different perspective.

Instead of giving a single answer, the system provides **multi-dimensional insights** such as career growth, market demand, risks, return on investment (ROI), and personal fit.

---

## 🎯 Problem Statement

In real life, decision-making is complex and involves multiple factors.
Most tools provide one-dimensional answers, which may not be sufficient.

👉 This system solves that by:

* Breaking decisions into multiple perspectives
* Providing structured and balanced insights
* Helping users make better informed decisions

---

## 🧠 System Architecture

The system consists of **5 AI Agents**, each responsible for a specific reasoning dimension:

| Agent                     | Role                                        |
| ------------------------- | ------------------------------------------- |
| 👨‍💼 Career Agent        | Analyzes career growth and opportunities    |
| 📊 Market Agent           | Evaluates industry trends and demand        |
| ⚖️ Devil’s Advocate Agent | Identifies risks and challenges             |
| 💰 ROI Agent              | Assesses time, effort, and financial return |
| 🧍‍♀️ Personal Fit Agent  | Matches decision with user interests        |

👉 A final decision is generated by combining all perspectives.

## ▶️ How to Run

